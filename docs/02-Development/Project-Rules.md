# Project Rules

## Project Overview

AssessmentAI is an AI-powered Assessment Platform designed specifically for educational institutions.

The goal is to simplify the complete assessment creation process while maintaining professional formatting and educational quality.

This repository currently contains only the clickable prototype.

No backend functionality should be implemented.

---

## Current Phase

Prototype Development

Technology

- HTML5
- CSS3
- Vanilla JavaScript

---

## General Rules

Always read the related Screen Specification before implementing any screen.

Never redesign an existing screen unless explicitly instructed.

Maintain consistency across all pages.

Reuse existing components.

Use dummy data only.

---

## Do Not

- Create backend
- Create APIs
- Create Database
- Add Authentication
- Add AI Integration
- Use React
- Use Vue
- Use Angular
- Use Bootstrap
- Use Tailwind
- Use TypeScript
- Use npm packages

---

## Development Workflow
