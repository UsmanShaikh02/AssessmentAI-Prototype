# Coding Standards

## HTML

Semantic HTML only.

Keep structure clean.

Comment major sections.

---

## CSS

Separate responsibilities.

variables.css

layout.css

components.css

style.css

Reuse classes.

Avoid duplicated CSS.

---

## JavaScript

Minimal.

No frameworks.

No unnecessary logic.

Use dummy data only.

---

## File Organization

Every page should be easy to understand.

Do not place all code into one file.

---

## Performance

Keep prototype lightweight.

Avoid unnecessary assets.

---

## Accessibility

Proper labels.

Keyboard friendly.

Readable contrast.

Meaningful button labels.
